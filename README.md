# PDF-to-Word-.docx-
Project Description: PDF to Word (.docx) Converter Using Flask and Python  This project is a simple and efficient web application built with Flask that enables users to convert PDF files into editable Word (.docx) documents. It provides a clean interface where users can upload a PDF, and then download the converted .docx file.  

The backend uses Python libraries for PDF parsing and DOCX generation, ensuring reliable extraction of text and basic formatting. The application handles file uploads securely by storing them temporarily and removing them after processing. Its modular structure makes it easy to maintain, extend, or integrate into larger systems.

Key Features:

Converts PDF files into Word (.docx) format

User-friendly upload and download workflow

Secure and temporary file handling

Lightweight Flask architecture

Good accuracy for text and simple layout extraction

This tool is suitable for developers exploring Flask, users who need quick PDF-to-Word conversion, or applications requiring automated document processing.
